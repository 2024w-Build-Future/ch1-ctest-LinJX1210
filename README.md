# 2024-Prerequisite-Experiments

## 使用教程

将远程仓库 clone 到本地或者使用 webIDE 进行实验

项目结构

```
.
├── exercises //所有习题都在此文件夹下
├── LICENSE
├── Makefile 
├── README.en.md
├── README.md
└── test //所有测例都在此文件夹下
```

本地测试

```
make test-output
```

会对所有题目进行测试。
